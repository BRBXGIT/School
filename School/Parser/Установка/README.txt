Сначала следует установить python, для этого запускаем python-3.10.1-amd64.exe предварительно скачав с сайта https://www.python.org/downloads/, и следуем инструкциям, везде соглашаемся и проставляем галочки. После дважды кликаем по файлу libraries.cmd(устанавливаем библиотеки, это делается только один раз). 

Для пользования программой нужно дважды кликнуть по файлу start.bat. После следует ввести ключевые слова, слова вводятся с маленькой буквы, программа автоматически сделает их заглавными. Список слов для поиска отображается после ввода каждого слова.
