Сначала следует установить python, для этого переходим по ссылке https://www.python.org/downloads/ и скачиваем python-3.10.1-amd64.exe, дальше запускаем файл и следуем инструкциям, везде соглашаемся и проставляем галочки. После дважды кликаем по файлу libraries.cmd(устанавливаем библиотеки, это делается только один раз). 

Для пользования программой нужно дважды кликнуть по файлу start.bat. После следует ввести ключевые слова, слова вводятся с маленькой буквы, программа автоматически сделает их заглавными. Список слов для поиска отображается после ввода каждого слова. 

После введиня всех нужных слов будет предложен выбор парсинга, если будет выбран медленный режим(slow) парсинг будет проводится по всем ссылкам на главной странице сайта, если будет введён quick парсинг будет только по новостной странице сайта.
